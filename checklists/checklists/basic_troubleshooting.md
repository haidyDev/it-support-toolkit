# Perus vianmäärityksen tarkistuslista

Tämä lista auttaa varmistamaan, että yleisimmät ongelmat tulee käytyä läpi ennen tiketin eskalointia.

## Tarkistuslista

- Onko laite käynnistetty uudelleen?
- Testattu eri verkkoa (WiFi / Ethernet / mobiilidata)?
- Toimiiko ongelma toisella käyttäjällä tai toisella laitteella?
- Onko käyttäjän tunnukset kunnossa (ei lukittu, salasana voimassa)?
- Onko järjestelmässä riittävästi levytilaa?
- Onko viimeisin päivitys tai ohjelma-asennus aiheuttanut muutoksen?
- Onko käyttäjä saanut virheilmoituksen? Kuvakaappaus tallennettu?

## Tavoite

Varmistaa, että yleisimmät ongelmat käydään läpi ennen syvempää teknistä tutkimista.
