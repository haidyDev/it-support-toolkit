# Verkko-ongelman selvittäminen – perusohje

Tämä ohje kuvaa vaiheittaisen prosessin tilanteeseen, jossa käyttäjällä ei ole verkkoyhteyttä.

## 1. Tunnista ongelma
- Ei yhteyttä lainkaan?
- Hidas yhteys?
- Yhteys katkeaa ajoittain?
- Ongelma vain tietyllä sivustolla tai sovelluksella?

## 2. Perustestit
- Käynnistä reititin / modeemi uudelleen (jos mahdollista)
- Irrota ja yhdistä verkko uudelleen
- Testaa eri laitteella

## 3. Paikallinen tarkistus
- Tarkista IP-osoite
- Tarkista DNS-asetukset
- Testaa yhteys esim. `ping` ja `nslookup`

## 4. Rajauksen tarkistus
- Onko ongelma vain käyttäjän laitteella?
- Onko muilla sama ongelma (verkko poikki yrityksellä / ISP:llä)?
- Onko palomuuri tai VPN estänyt yhteyden?

## 5. Dokumentointi
- Kerää virheilmoitukset
- Merkitse, missä vaiheessa ongelma ilmenee
- Kuvaa, mitkä testit on jo tehty

Tämän ohjeen tarkoitus on nopeuttaa vianmääritystä ja auttaa ongelman rajaamisessa.
